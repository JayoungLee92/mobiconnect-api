API Methods
GET findAll : Provides list of all available offers

GET findAll

Request

Name Type Required Value Description
api_key String Y unique key The key provided for publishers to call API.
offer_id String N numbers (comma delimited) The Offer's unique ID. Multiple inputs are available.
require_approval String N enabled / disabled Whether or not Offers require approval to run.
payout_start String N numbers (includes up to second decimal place) Minimum payout of the Offers.
payout_end String N numbers (includes up to second decimal place) Maximum payout of the Offers.
cap_enabled String N enabled / disabled Whether or not offers have maximum number of conversions allowed. If value is set to "disabled" then there is no cap.
http://publisher.mobiconnect.co.kr/publisher/openapi/findAll?api_key={YOUR_API_KEY}&offer_id={}&require_approval={}&payout_start={}&payout_end={}&cap_enabled={}

Response

Name Type Value Example value Description
offer_name String - testoffer KR AOS Non The Offer's name.
status String active active The status of the Offer. Publishers can only view active Offers.
offer_id String numbers 5090 The Offer's unique ID.
expires_date String yyyy-mm-dd hh:mm:ss 2018-12-31 23:59:59 Expiration date for the Offer.
category String - Game_MainStream The category used by MobiConnect to define the Offer.
preview_url String - https://play.google.com/store/apps/details?id=com.mctestoffer The final destination URL of the offer. Allows Publishers to view sample landing page.
default_offer_url String - http://tracking.mobidays.com/adserver/track/clickApi?offer_id=95513&pub_id=9999 The default URL of the offer. Often used to place click url.
description String - KPI / Deduction : D+1 Retention 10% and below (sub-pub 30 above)\nOS version : Android 4.0.3 or later\nTraffic : Strictly non-incent traffic only/Direct traffic preferred/No SNS traffic/No P2P The description of the Offer.
currency String USD USD Currency symbol for the Offer.
payout_type String cpa_flat cpa_flat Specifies the method of calculating payout for this Offer.
payout_per_conversion String numbers (includes up to second decimal place) 1.9 The flat rate/amount paid for this Offer.
require_approval String enabled / disabled Disabled Whether or not Offers require approval to run.
cap_enabled String enabled / disabled Enabled Whether or not offers have maximum number of conversions allowed. If value is set to "disabled" then there is no cap.
daily_conversion_cap String numbers 1000 Maximum number of conversions allowed per day for this Offer. If value is set to "0" then there is no cap.
monthly_conversion_cap String numbers 5000 Maximum number of conversions allowed per month for this Offer. If value is set to "0" then there is no cap.
country_code String - (comma delimited) KR,CN,US Targeted geo allowed for the Offer.
creatives_name String - Test Creatives The name of the creatives.
creatives_file_name String file name Test_Creatives.zip The name of the file attached for creatives.
creatives_file_url String - https://mobione.s3.amazonaws.com/Test_Creatives.zip The download URL for creative files attached.
