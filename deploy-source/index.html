API Index

MobiConnect API
Provides a list of available offers and authorization for offer request.
All requests require unique API Key provided for publishers.
The API Key can be found or reissued from API KEY page of Publisher Site.
API Version
API v2 : The latest version of API (updated at - 2018.07.31)
API v1 : For newly integrating partners, please use API v2 to benefit from future updates. API v1 is outdated.
