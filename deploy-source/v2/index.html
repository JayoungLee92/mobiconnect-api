API v2( Updated at - 2018/07/31 )

API URL
http://api.mobiconnect.co.kr/v2/publisher/
API Methods
GET /offers : Returns all available offers.
GET /offers/{offerId} : Returns one or more offers by their IDs.
POST /offers/{offerId}/approval : Requests approval for one or more offers by their IDs. Returns approval status of the offer if already in request.

GET /offers

Request

Name Type Required Value Description
api_key * STRING Y unique key The key provided for publishers to call API.
os String N 0/1/2 The Offer's OS type.
0 - iOS
1 - AOS (Android)
2- ETC
categories STRING ARRAY(comma delimited) N Category list The category used by MobiConnect to define the Offer.
countries STRING ARRAY(comma delimited) N Country list Targeted geo allowed for the Offer.
payout_start NUMBER(20,2) N numbers (x.xx) Minimum payout of the Offers.
payout_end NUMBER(20,2) N numbers (x.xx) Maximum payout of the Offers.
cap STRING N 0 / 1 Whether or not offers have maximum number of conversions allowed. If value is set to "0" then there is no cap.
0 - CAP open
1 - CAP enabled
approval_type STRING N 0 / 1 Whether or not Offers require approval to run.
0 - Requires approval : false
1 - Requires approval : true
approval_status STRING N 1 / 2 / 3 / 4 The approval status of the offer.
1 - unapproved : Offer is available for approval request
2 - pending : Request for approval is on review
3 - approved : Offer has been approved
4 - rejected : Request for approval has been rejected
